# KNIFE Quick Start

KNIFE is a keyboard-first fractal explorer. You can use mouse and touch too, but the fastest way to navigate is with the keyboard.

Need full depth after this card?
- [Open the full Pilot Manual](MANUAL.md)
- [Open the exact binding matrix](REFERENCE.md)

## Start here

- Press `?` to open the full input map and control reference.
- Use the arrow keys to pan around.
- Use `=` to zoom in and `-` to zoom out.
- Press `r` to recenter the view.
- Press `p` to open or cycle palettes.
- Press `f` to open the fractal menu.
- Press `d` for detail controls.
- Press `s` for speed and palette-cycling controls.
- Press `i` to open the info panel.
- Press `e` to view the current fractal equation.

## Basic navigation

### Keyboard
- Pan: arrow keys
- Zoom in: `=`
- Zoom out: `-`
- Recenter: `r`
- Help: `?`

### Mouse
- Pan: click and drag
- Zoom: mouse wheel
- Click the image to zoom in
- `Shift+click` the image to zoom out

### Touch
- Pan: one-finger drag
- Zoom: pinch/spread
- Optional on-screen nav pad: press `n`

## Most useful controls

- `p` = palette menu / cycle palette
- `f` = fractal menu
- `d` = detail menu
- `s` = speed menu
- `o` = switch colorizer mode
- `m` = toggle hi-res rendering
- `i` = toggle info panel
- `?` = open help/input map

## If the screen feels too busy or too sparse

- `k` toggles compact header mode
- `h` toggles header-only mode
- `Shift+h` hides all UI
- Long-press the UI restore hotspot to bring everything back if needed

## Good first-time workflow

1. Press `?` and skim the input map.
2. Use arrow keys and `=` / `-` to explore.
3. Press `p` and try a few palettes.
4. Press `f` and switch fractal types.
5. Press `i` to open the info panel and read the current context.
6. Press `e` to inspect the equation behind the current fractal.

## A note on design

KNIFE was built around fast keyboard use first, with mouse and touch added to stay as close to feature parity as practical. Some advanced actions still remain keyboard-only.
